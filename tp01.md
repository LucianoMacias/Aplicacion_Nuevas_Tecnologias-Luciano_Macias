# 1. Netflix
## Problema que resuelve con ML

### Netflix utiliza Machine Learning para recomendar películas y series que probablemente interesen a cada usuario, mejorando la experiencia de uso y aumentando el tiempo de visualización.

## Tipo de ML utilizado

**Aprendizaje Supervisado (Supervised Learning)**

**Justificación:**
### El sistema aprende a partir de datos históricos donde conoce las acciones realizadas por los usuarios (ver una serie, abandonarla, calificarla, etc.). Con esos ejemplos puede predecir qué contenido podría gustarle a cada persona.

**Datos necesarios**
- Historial de visualización.
- Tiempo de reproducción.
- Calificaciones dadas por los usuarios.
- Géneros preferidos.
- Datos demográficos básicos.

# 2. Google Translate

## Problema que resuelve con ML

### Permite traducir texto entre diferentes idiomas de manera automática.

## Tipo de ML utilizado

**Aprendizaje Supervisado (Supervised Learning)**

**Justificación:**
### Los modelos son entrenados utilizando millones de pares de textos ya traducidos por humanos. El sistema aprende la relación entre frases en distintos idiomas para generar nuevas traducciones.

**Datos necesarios**
- Textos en múltiples idiomas.
- Traducciones realizadas por profesionales.
- Correcciones de usuarios.
- Diccionarios y corpus lingüísticos.

# 3. Spotify

## Problema que resuelve con ML

### Recomienda canciones, artistas y listas de reproducción personalizadas para cada usuario.

##Tipo de ML utilizado

**Aprendizaje Supervisado y No Supervisado**

**Justificación:**

- Supervisado: aprende de los gustos previos de los usuarios.
- No supervisado: agrupa canciones similares según características musicales sin necesidad de etiquetas previas.

### Si se debe elegir uno solo, puede clasificarse principalmente como Supervisado, ya que las recomendaciones se basan en comportamientos históricos de los usuarios.

**Datos necesarios**
- Canciones escuchadas.
- Tiempo de escucha.
- Canciones omitidas.
- Playlists creadas.
- Géneros musicales.
- Información de artistas.

# 4. Tesla Autopilot

## Problema que resuelve con ML

### Ayuda a conducir el vehículo identificando carriles, señales de tránsito, peatones y otros vehículos.

## Tipo de ML utilizado

**Aprendizaje Supervisado** 

**Justificación:**
### Los modelos son entrenados con millones de imágenes y videos previamente etiquetados donde se indica qué objetos aparecen en cada situación.

**Datos necesarios**
- Imágenes de cámaras.
- Videos de conducción.
- Información de sensores.
- Posición del vehículo.
- Etiquetas de objetos (autos, peatones, señales, etc.).

# 5. Mercado Libre (Argentina / Latinoamérica)

## Problema que resuelve con ML

### Utiliza Machine Learning para recomendar productos, detectar fraudes y ordenar resultados de búsqueda.

## Tipo de ML utilizado

**Aprendizaje Supervisado**

**Justificación:**
### El sistema aprende a partir de compras anteriores, clics, búsquedas y casos de fraude ya identificados para realizar predicciones sobre futuros comportamientos.

**Datos necesarios**
- Historial de compras.
- Búsquedas realizadas.
- Productos visitados.
- Datos de transacciones.
- Reportes de fraude.
- Calificaciones de vendedores y compradores.
